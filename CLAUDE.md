# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

All commands run from the `client/` directory:

```bash
npm run dev       # Start dev server on http://localhost:3000
npm run build     # Production build
npm run preview   # Preview production build
npm run test      # Run Vitest tests
npm run lint      # Biome linter
npm run format    # Biome formatter
npm run check     # Full Biome check (lint + format)
```

To run a single test file: `npx vitest run src/path/to/file.test.tsx`

## Stack

- **Framework**: TanStack Start (React 19) with SSR via Nitro
- **Routing**: TanStack Router — file-based routing under `client/src/routes/`
- **State**: TanStack Store (global) + TanStack Form (form state)
- **Styling**: Tailwind CSS 4 (configured via Vite plugin)
- **Linting/Formatting**: Biome (replaces ESLint + Prettier)
- **Testing**: Vitest + Testing Library
- **AI**: TanStack AI with Anthropic Claude (`ANTHROPIC_API_KEY` in `.env.local`)

## Architecture

### Routing
TanStack Router uses file-based routing. Adding a file to `client/src/routes/` auto-registers it. **Never manually edit `client/src/routeTree.gen.ts`** — it's auto-generated by the Vite plugin on `npm run dev`.

- `__root.tsx` — root HTML shell and layout wrapper
- `index.tsx` — home route (`/`)
- Server functions (backend logic) live in route files using `createServerFn`

### Path Aliases
Both `#/*` and `@/*` resolve to `client/src/*`.

### Design System
- Primary/accent: Gold `#B5A167`
- Secondary/text: Navy `#003366`
- Background: White `#FFFFFF`, Cream `#F5F3EC`
- Icons: Lucide React

## Application Spec

This is a budget planning PWA with five main screens (see `client/docs/budget-tracker-spec.md` for full details):

1. **Setup** — Budget creation wizard (name, start month)
2. **Income Planning** — Monthly income sources in a 12-month spreadsheet grid
3. **Expense Planning** — Expense categories with monthly breakdown grid
4. **Category Management** — CRUD + drag-to-reorder categories with icons
5. **Overview Dashboard** — KPI metric cards, bar chart (income vs expenses), category breakdown, cash flow table

Key shared components: `BudgetGrid` (editable spreadsheet), `EditableCell` (gold focus border), `TopNav` (tabs across all screens).
